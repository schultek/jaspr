# embed

Signature of the embed component:

```dart
const embed({
  required String src,
  String? type,
  int? width,
  int? height,
  String? id,
  String? classes,
  Styles? styles,
  Map<String, String>? attributes,
  Map<String, EventCallback>? events,
  Key? key,
})
```

Example usage:

```dart
embed(src: '...')
```
