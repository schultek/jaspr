# link

Signature of the link component:

```dart
const link({
  required String href,
  String? rel,
  String? type,
  String? as,
  String? id,
  String? classes,
  Styles? styles,
  Map<String, String>? attributes,
  Map<String, EventCallback>? events,
  Key? key,
})
```

Example usage:

```dart
link(href: '...')
```
